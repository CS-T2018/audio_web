About This Project

The Music Source Separation App can be used to separate a song into four components: vocals, bass, drums, and others. OpenUnMix is one of many music separation models that can be used on pop music to separate one song into those four components.

We have provided 5 different audio files on which to test the functionality of the website. The user can listen to the original file before and after separation. Clicking the separate button will perform the separation on the GPU workstation. The user will be redirected to a new page to view the separation results when the process is complete.

It must be noted that this process is slow and computationally expensive and, as a result, can only be performed on short audio files.
